概述

这次要实现 while 循环的处理，同时加上 continue 和 break 。有了前面 if-else 和短路求值的尝试，这里就很简单了。

处理 while

增加了一条规则 Stmt ::= "while" "(" Exp ")" Stmt，只需要：

  1. 词法分析中加入新关键字 while
  2. 语法分析的 MatchedStmt 中加入新的规则 "while" "(" Exp ")" Stmt
  3. 语义分析为新类型节点加个函数
  4. IR 生成在 while 节点处仿照 if-else 构建 entry-body-end 的循环语句块
  5. RISC-V 生成不用改

即可。

语义分析

要求 break 和 continue 只能用于循环语句内，在语义分析中增加 int while_levels 记录 while 循环嵌套的层数 (注意：单纯的 bool 不行，处理嵌套 while 时会出错)，遇到 break 和 continue 时检查该变量即可。

IR 生成

这两条语句都可以用简单的跳转表示：

  • break 会跳转到当前 while 循环的结尾，也就是 while_end
  • continue 会跳转到当前 while 循环的开头，也就是 while_entry

由于 while 结构是嵌套的，需要添加数据结构来记录各层 while 循环的标签名，以便让 break 和 continue 获取当前所在 while 的标签，这里选择用栈。
